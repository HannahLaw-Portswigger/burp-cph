Target Specific Matches
when an expression is found multiple times in a message

Available targeting options within step 2 of the Parameter handling pane are:

  1. Target the first of the matches
  2. Target each of the matches (i.e., all of them)
  3. Target a specific subset of matches

When targeting a subset of matches, indices and ranges can be used. E.g.:

  • Target the 1st, 4th, 6th and 8th matches: 0,3,5,7
  • Target the first 7 matches but not the 8th match: 0:7
  • Target the first 7 matches and the 10th match: 0:7,9
  • Target the last and penultimate matches: -1,-2
  • Target all but the last match: 0:-1
