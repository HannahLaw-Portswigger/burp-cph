Scope CPH Tab
CPH tabs will only modify requests matching their configured scope

To modify the scope of a CPH tab, it is possible to:

  1. Choose between requests, responses or both
  2. Match a specific expression, in addition to the above

Please note that any part of the HTTP message, including verbs, status codes and headers, can be used for expression matching
